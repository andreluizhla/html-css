Jeito Certo:

O jeito certo seria colocar os estilos seria colocar a tag style dentro da tag head

Então coloque os parâmetros dentro com as chaves "{...}" e dentro, os atributos do "style".

Exemplo: 


        [...]
        style
            h1 {
                color: amarelo;
            }
            ...
        /style
        [...]
