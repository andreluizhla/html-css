Formatos de imagens

Para imagens no site:

.jgp (JPEG)
É um formato bom para deixar durante o site, tendo um tanhanho bem reduzido, mas fica distorcido de perto
.png
É um formato um pouco maior de imagem, porém esse formato fica transparente, o que pode ser bom

Para Favicons

.ico
Qualquer navegador reconhece, e o mais utilizado
.png
Olha o PNG de novo, sendo um formato transparente seria meio óbvio que estaria aqui
.svg
Esse formato é Vetorizado, sendo a melhor opção, porém nem todos os navegadores reconhece esse formato
